# writer_journal
full stack personal journal app.

Stack:

Frontend: React, HTML/CSS

Backend: Spring Boot + JPA + Postgresql

Current Features:
1. Submit Journal Entries
2. Database of Entries
3. Scroll through old Entries
4. AI Writing Prompts

Future Features:
1. Log in
2. Delete Entries
